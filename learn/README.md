# Learn LEZ

Start here if you are new to the Logos Execution Zone. These guides cover the protocol from the ground up.

| File | What it covers |
|------|----------------|
| `protocol.md` | Architecture overview — what LEZ is and how it fits in the Logos stack |
| `accounts.md` | Public and private accounts, keys, commitments, nullifiers |
| `execution.md` | Public vs private execution, Risc0, ZKPs, what gets proven |
| `programs.md` | How to write and deploy LEZ programs using lez-framework and SPEL |
| `tools.md` | Ecosystem tooling — what exists and when to use it |
