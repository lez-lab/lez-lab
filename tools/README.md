# Tools

Dev tools for exploring, building, and interacting with LEZ.

| Tool | Description | Source |
|------|-------------|--------|
| `spel/` | SPEL language for LEZ programs | [logos-co/spel](https://github.com/logos-co/spel) |
| `lez-framework/` | #[lez_program] macro and framework | [logos-co/lez-framework](https://github.com/logos-co/lez-framework) |
| `logos-scaffold/` | CLI to bootstrap a LEZ environment | [logos-co/logos-scaffold](https://github.com/logos-co/logos-scaffold) |
| `lez-code-explorer/` | Interactive LEZ code explorer | [KindaInsecureBot/lez-code-explorer](https://github.com/KindaInsecureBot/lez-code-explorer) |
