# Programs

Ecosystem programs built on LEZ — official and community. Each folder covers what the program does, links to the source, and shows example usage where available.

| Program | Description | Source |
|---------|-------------|--------|
| `lez-programs/` | Official core programs | [logos-blockchain/lez-programs](https://github.com/logos-blockchain/lez-programs) |
| `lez-multisig/` | Multisig using lez-framework | [logos-co/lez-multisig](https://github.com/logos-co/lez-multisig) |
| `payment-streams/` | Payment streams protocol | [logos-co/logos-lez-payment-streams](https://github.com/logos-co/logos-lez-payment-streams) |
| `lez-rln/` | RLN spam protection registration | [logos-co/logos-lez-rln](https://github.com/logos-co/logos-lez-rln) |
| `eth-lez-atomic-swaps/` | ETH/LEZ atomic swaps | [logos-co/eth-lez-atomic-swaps](https://github.com/logos-co/eth-lez-atomic-swaps) |
| `spelbook/` | On-chain program registry | [jimmy-claw/spelbook](https://github.com/jimmy-claw/spelbook) |
| `lezard/` | Framework for building/testing LEZ programs | [gravityblast/lezard](https://github.com/gravityblast/lezard) |
| `music-escrow/` | Atomic music distribution on LEZ | [jimmy-claw/music-escrow](https://github.com/jimmy-claw/music-escrow) |
